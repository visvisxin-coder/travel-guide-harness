# Visual Style Guide

The default visual language is clean editorial travel planning.

## Principles

- Information first.
- Route logic should be visible at a glance.
- Use restrained color and strong spacing.
- Avoid noisy decoration.
- Use cards only for repeated items like stops or day summaries.

## Recommended Components

- Cover
- Trip overview
- Daily timeline
- Place card
- Budget note
- Transport tips
- Backup plan block
